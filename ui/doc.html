About response_mode

  • fragment: 
     HTTP Redirect to https://redirect_uri#code=xxxx

    Beacuse fragment is not included with redirection based on http protocol, front end javascript must be used to forward fragment to backend if needed.

  • query: 
     HTTP Redirect to https://redirect_uri?code=xxxx
  • form_post: 
    html file with hidden form automatically submitted by javascript.
